Hexo on Github--Custom Domain Tutorial

First post dedicated to teach my darling about setting custom domains~

  • I haven’t figured out how to deal with the encoding problem (Chinese doesn’t render yet) so this will be in english.

Before you begin

Make sure the blog pages works properly on github pages

Click on “Settings”:

Click on the link after “Your site is published at: “ and make sure the generated hexo page shows up:

Create the CNAME file

In the “source” folder under your hexo blog directory, create a new file named “CNAME”.
Open it and edit the content to be the your domain—no http, no www., just what you signed up for on GoDaddy:

Save and run


And you should now have this CNAME file in your github repository root:

Change DNS settings on GoDaddy

Sign into your GoDaddy account, click on “Domains” then “Launch” on your desired domain:

Under “DNS Zone file”

Remove the one record under A(Host) section. Then click on “Add record”

Selct the record type A:

Then fill in the record as follows:

When done with this one, enter another record with the same info except use 192.30.252.154 in the “points to” field.
(Don’t forget to save the changes as prompted)

By this point the setup should be done. You’ll probably have to wait for an hour or two before the DNS changes take effect (I can go into details about this if you want clarification but gah, technical). When it’s done, you should see the blog through your custom domain!
